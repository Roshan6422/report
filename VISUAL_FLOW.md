## Real-Time Rotation Formula

```python
# For each translation batch:
shift = (category_number * 100 + batch_index) % total_engines
selected_engine = engines[shift]

# Example with 6 engines:
Cat 01, Batch 0: (1*100 + 0) % 6 = 4 вҶ’ Engine 4
Cat 01, Batch 1: (1*100 + 1) % 6 = 5 вҶ’ Engine 5
Cat 02, Batch 0: (2*100 + 0) % 6 = 2 вҶ’ Engine 2
Cat 02, Batch 1: (2*100 + 1) % 6 = 3 вҶ’ Engine 3
```
