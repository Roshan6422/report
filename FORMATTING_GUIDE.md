# Security Situation Report - 100% Formatting Guide

## Overview

This document describes the EXACT formatting specifications used in the official Security Situation Report, matching the sample dated 17th-18th March 2026.

---

## Page Layout

### Paper & Margins
- **Paper Size:** A4 (210mm × 297mm)
- **Margins:** 25mm (top, left, right), 20mm (bottom)
- **Font:** Times New Roman throughout
- **Base Font Size:** 11pt
- **Line Height:** 1.15 (single spacing with slight increase)

---

## Header Section (Page 1)

### Page Number
- **Position:** Top right
- **Font:** Times New Roman, 11pt, normal weight
- **Spacing:** 8mm below page number

### Confidential Label
- **Text:** "Confidential"
- **Font:** Times New Roman, 14pt, bold
- **Alignment:** Center
- **Spacing:** 3mm below

### Logo
- **Size:** 28mm width (height auto-scales)
- **Alignment:** Center
- **Spacing:** 3mm below

### IG's Address
- **Text:** "IG's Command / Information Division, Mirihana, Sri Lanka."
- **Font:** Times New Roman, 11pt, bold
- **Alignment:** Center
- **Line Height:** 1.2
- **Spacing:** 0mm below (tight)

### IG's Title
- **Text:** "Inspector General of Police."
- **Font:** Times New Roman, 11pt, bold
- **Alignment:** Center
- **Spacing:** 5mm below

### Report Title
- **Text:** "Security Situation Report"
- **Font:** Times New Roman, 18pt, bold
- **Alignment:** Center
- **Letter Spacing:** 0.3pt
- **Spacing:** 3mm above, 2mm below

### Date Range
- **Format:** "From 0400 hrs. on 17<sup>th</sup> March 2026 to 0400 hrs. on 18<sup>th</sup> March 2026"
- **Font:** Times New Roman, 11pt, bold
- **Alignment:** Center
- **Decoration:** Underline
- **Spacing:** 5mm below

---

## Section Headers

### Main Section Headers
- **Format:** "01. SECTION NAME:"
- **Font:** Times New Roman, 11pt, bold
- **Alignment:** Left
- **Spacing:** 3mm above, 3mm below
- **Case:** UPPERCASE

### "Nil" Sections
- **Format:** "01. SECTION NAME: Nil"
- **Same styling as main headers**

---

## Province Headings

### Format
- **Text:** "S/DIG  PROVINCE_NAME PROVINCE"
- **Font:** Times New Roman, 11pt, bold
- **Alignment:** Center
- **Letter Spacing:** 0.5pt (slightly wider)
- **Spacing:** 4mm above, 3mm below
- **Case:** UPPERCASE

### Example
```
S/DIG  SABARAGAMUWA PROVINCE
```

---

## Incident Layout (Two-Column)

### Left Column (DIG Side)
- **Width:** 28% of page width
- **Vertical Alignment:** Top
- **Font:** Times New Roman, 11pt, bold
- **Line Height:** 1.15
- **Padding Right:** 3mm

#### DIG District Line
- **Format:** "DIG [District Name] District"
- **Font Weight:** Bold
- **Spacing:** 0mm below

#### Division Line
- **Format:** "[Division Name] Div."
- **Font Weight:** Bold
- **Font Style:** Italic
- **Spacing:** 0mm above

### Right Column (Body Side)
- **Width:** 72% of page width
- **Vertical Alignment:** Top
- **Font:** Times New Roman, 11pt, normal weight
- **Line Height:** 1.15
- **Text Alignment:** Justify

#### Station Name
- **Format:** "STATION_NAME:"
- **Font Weight:** Bold
- **Case:** UPPERCASE

#### Incident Title (Summary)
- **Format:** "(Title text)"
- **Font Weight:** Bold
- **Placement:** Immediately after station name

#### Body Text
- **Font Weight:** Normal
- **Follows incident title**

#### Reference Code
- **Format:** "(CTM.XXX)" or "(OTM.XXX)"
- **Placement:** End of body text
- **Font Weight:** Normal

### Complete Example
```
DIG Ratnapura          EMBILIPITIYA: (Arrest of suspects along with a detonator and
District               gunpowder) On the 17th March 2026, acting on an information
                       received through the 1-1-9 project, police arrested...
Embilipitiya Div.      (OTM.1421)-A
```

### Spacing Between Incidents
- **Margin Bottom:** 3mm

## Signature Section

### "Other Matters" Header
- **Text:** "04. OTHER MATTERS OF INTEREST AND IMPORTANCE: Nil"
- **Font:** Times New Roman, 11pt, bold
- **Spacing:** 10mm above

### Signature Placeholder (Prepared By)
- **Style:** Underline with italic text "Signature placeholder"
- **Spacing:** 5mm above, 3mm below

### Prepared By Text
- **Format:**
  ```
  Prepared by: PS 51258
            U.G. Ajith Priyantha
  ```
- **Font:** Times New Roman, 11pt, normal
- **Indentation:** 10 spaces for name line
- **Spacing:** 15mm below

### Signature Placeholder (Checked By)
- **Style:** Underline with italic text "Signature placeholder"
- **Alignment:** Center
- **Spacing:** 10mm above, 3mm below

### Checked By Text
- **Format:**
  ```
  Checked by: B.A. Sujith Ganganatha
  Chief Inspector of Police
  Duty Officer
  IG's Command & Information Division,
            Mirihana
  18th March 2026
  ```
- **Font:** Times New Roman, 11pt, normal
- **Line Height:** 1.3
- **Indentation:** 10 spaces for "Mirihana" line

## Typography Details

### Superscript for Ordinals
- **Usage:** Date ordinals (st, nd, rd, th)
- **Font Size:** 0.7em (70% of base)
- **Vertical Align:** Super
- **Line Height:** 0 (no line height impact)
- **Example:** 17<sup>th</sup> March 2026

### Bold Text Usage
- All section headers
- Province headings
- Station names
- Incident titles (in parentheses)
- DIG/Division information
- Signature section labels
- Distribution list title

### Italic Text Usage
- Division names (e.g., "Embilipitiya Div.")
- Signature placeholders

### Underline Usage
- Date range in header
- Signature placeholder lines

## Differences from General Report

The Security Situation Report differs from General Situation Report in:

1. **Title:** "Security Situation Report" vs "General Situation Report"
2. **Sections:** 3 sections vs 10 sections
3. **Logo:** security_logo.png vs police_logo.png
4. **Focus:** Security matters only (arms, subversive activities)

All formatting specifications remain identical.
