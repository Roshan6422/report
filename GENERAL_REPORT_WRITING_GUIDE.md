# General Situation Report - Writing Format Guide

## Overview
This guide defines the EXACT writing format for General Situation Report incidents, based on the official sample.

---

## Writing Format Requirements

### 1. Station Name (UPPERCASE, Bold)
- Format: `STATION_NAME:`
- Example: `PUSSELLAWA:`

### 2. Incident Summary (Bold, in parentheses)
- Brief description of the incident type
- Format: `(Summary text)`
- Example: `(A case of a burglary...)`

### 3. Detailed Body Text (100-300 words)

Must include ALL of the following details:

#### Crime Details
- Exact amounts stolen (cash with Rs. symbol)
- Specific quantities (gold jewellery with sovereigns)
- Exact valuations (Rs. amounts)
- Method of crime (e.g., "by breaking and entering through a window")

#### Time & Date
- Exact time range with "hrs" format
- Full dates with ordinal suffixes (14th, 16th)
- Format: "between 0900 hrs on 14th of March 2026 and 1400 hrs on 16th of March 2026"

#### Location
- Complete address with area names
- Format: "at [Location], [Area]"
- Example: "at Mawelakanda, Maswela"

#### Complainant Information
- Full name with initials
- Telephone number in parentheses
- Format: "Complainant named [Name], (TP [phone])"
- Example: "H. S. Kandegedara, (TP 072-5340753)"

#### Suspect Information
- Name if known, or "Unknown"
- Age if available
- Address if available
- Format: "Suspect: [Name/Unknown]"
- Example: "Suspect: Unknown"

#### Recovery Status
- Status of stolen items
- Format: "The stolen [items] not recovered and investigations in process"
- Example: "The stolen cash and jewellery not recovered and investigations in process"

#### Motive
- Clear statement of motive
- Format: "Motive: [reason]"
- Example: "Motive: For illegal gain"

#### Reference Code
- At the end in parentheses
- Format: `(CTM.XXX)` or `(OTM.XXX)`
- Example: `(CTM.522)`

---

## Complete Sample Format

```
PUSSELLAWA: (A case of a burglary...) A case of a burglary of Rs. 150,000/= 
and gold jewellery (01 ¾ sovereigns) valued Rs. 600,000/= by breaking and 
entering through a window of a house was reported to the police station. The 
offence took place between 0900 hrs on 14th of March 2026 and 1400 hrs on 
16th of March 2026 at Mawelakanda, Maswela. Complainant named H. S. 
Kandegedara, (TP 072-5340753). Suspect: Unknown. The stolen cash and 
jewellery not recovered and investigations in process. Motive: For illegal 
gain. (CTM.522)
```

---

## Key Differences from Security Report

1. **More detailed crime specifics** - exact amounts, valuations
2. **Complainant details** - name and phone number
3. **Recovery status** - explicit statement
4. **Motive statement** - always included
5. **Longer narratives** - typically 150-250 words

---

## Writing Rules

1. Use professional police terminology
2. Include ALL numerical details (amounts, quantities, times)
3. Use complete addresses with area names
4. Always include complainant information
5. State recovery status explicitly
6. End with motive and reference code
7. Use "hrs" for time format (not "hours")
8. Use ordinal suffixes for dates (14th, not 14)
9. Use Rs. symbol for currency
10. Use fractions for gold quantities (01 ¾ sovereigns)
