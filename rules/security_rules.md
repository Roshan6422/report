# Security Report Writing Rules
# ආරක්ෂක වාර්තා ලිවීමේ නීති

> [!IMPORTANT]
> **CRITICAL: INSTITUTIONAL FIDELITY**
> Security reports must follow a specific administrative narrative. Include all OTM/AR references exactly as they appear.

1. **Focus**: Arrests of tri-forces/police, recoveries of weapons/explosives, subversive activities, and narcotics (large scale).
2. **Context**: Use formal, administrative language for institutional records.
3. **Accuracy**: Precise location details (e.g. "DIG Ratnapura District, Embilipitiya Div.") are essential.
4. **Style Guidelines**:
   - Use "On the [Date], [Action] ..." as the standard opening.
   - Mention the specific bureau (e.g. Police Narcotics Bureau) if applicable.
   - Include the OTM reference in parentheses at the end: `(OTM.XXXX)-X`

---

### 📝 EXAMPLE TRANSLATION (SECURITY STYLE) - FOLLOW THIS EXACT FORMAT
**English Output Style:**
"EMBILIPITIYA: On the 17th March 2026, acting on an information received through the 1-1-9 project, police arrested a Buddhist monk named Rev. Embilipitiye Indrarathana thero, aged 68, the chief incumbent of the Sri Darshanagiri viharaya, Darshanagama, Embilipitiya and another person named N.U. Samanchandra, aged 56 of # 896, Mayuragama, Sewanagala along with an electric detonator and 80g of gunpowder, while digging a tunnel with the intention of treasure hunting in the temple premises. The suspects are scheduled to be produced before the Magistrate court, Embilipitiya on the 18th March 2026. (OTM.1421)-A"

---

> [!CAUTION]
> **NO REPETITION ALLOWED**
> Do not repeat the same incident description or any phrases. If you have nothing new to say, stop. Never loop the text.
