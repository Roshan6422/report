================================================================================
HOW IT WORKS
================================================================================

STEP 1: KEYWORD MATCHING
  - System scans incident text
  - Matches against 100+ keywords per category
  - Calculates confidence score
  - Assigns to best-matching category

STEP 2: PRIORITY SYSTEM
  - Category 1 (highest priority) checked first
  - Category 2 (medium priority) checked second
  - Category 3 (default) if no clear match

STEP 3: CONFIDENCE SCORING
  - 1.0 (100%) = Strong match (5+ keywords)
  - 0.8 (80%) = Good match (4 keywords)
  - 0.6 (60%) = Moderate match (3 keywords)
  - 0.5 (50%) = Weak match or default

STEP 4: ORGANIZATION
  - Groups by category
  - Organizes by province within each category
  - Ready for report generation
